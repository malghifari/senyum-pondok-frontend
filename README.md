# senyum-pondok-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
Change local .env
```
npm run serve
```

### Deployment
```
npm run build
```
Copy file in deploy package to /dist
Change staging .env
```
npm run staging
```
Change production .env
```
npm run deploy
```
```
git push origin master
```
